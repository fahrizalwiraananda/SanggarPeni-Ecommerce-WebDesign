Pembayaran Via BRI

Lakukan pembayaran pesanan melalui bank BRI dengan menggunakan nomor virtual akun di bawah ini :

Maks. :

02 Agustus 2022, 11:00 WIB

Total Tagihan :

Rp. 178.300

Cara Bayarnya Bagaimana ?

  • Pilih menu “Transaksi Lain”
  • Selanjutnya pilih menu “Lainnya”
  • Kemudian pilih menu “Pembayaran”
  • Tekan pada pilihan “BRIVA”
  • Masukkan Nomor BRI Virtual Account (Contoh 26215-xxxxxxxxxxxx) lalu tekan “Benar”
  • Selanjutnya muncul konfirmasi pembayaran, tekan “Ya” jika semua data sudah sesuai
  • Transaksi berhasil dilakukan

  • Buka website internet banking BRI
  • Login dengan memasukkan User ID dan Password
  • Pilih pada menu “Pembayaran”
  • Selanjutnya pilih menu “BRIVA”
  • Kemudian pilih rekening Pembayar
  • Masukkan Nomor Virtual Account BRI Anda (Contoh 26215-xxxxxxxxxxxx)
  • Lalu, masukkan nominal yang akan dibayarkan
  • Masukkan password dan Mtoken Anda
  • Transaksi berhasil dilakukan

  • Login ke mobile banking
  • Tekan menu “Pembayaran”
  • Kemudian pilih menu “BRIVA”
  • Masukkan nomor virtual account BRI Anda beserta jumlah pembayaran
  • Masukkan nomor PIN Anda
  • Tekan “OK” untuk melanjutkan transaksi
  • Transaksi berhasil dilakukan
  • Setelah beberapa saat, Anda akan menerima SMS konfirmasi
